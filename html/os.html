Operating System


Also known as OS, the most common OS in the market is Windows and MacOS. Linux on the other hand, is a free open-sourced OS that you can find online.

Windows OS

  • Comes in a few varities, Windows (7/8/10).
  • As Windows is widely used worldwide, software are more easily avaliable.
  • Normal security level.

MacOS

  • Only comes together with Apple products only
  • Some limitations to softwares compatibility
  • High in Security
  • Hybrid System if you have other Apple devices

Linux OS

  • Free Operating System
  • Quite a sum of restrictions on softwares compatibility
  • Security at risk, as Linux is developed by developers worldwide.
